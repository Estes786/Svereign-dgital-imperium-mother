## 🎯 WHAT IS SMA?

### Sovereign Multi-Industry Agent (SMA)
SMA adalah **meta-platform** — platform di mana semua "Sovereign Agents" (SICA, SHGA, SCA, dan agents baru) hidup dan bisa dikombinasikan oleh user sesuai kebutuhan industri mereka.

```
ANALOGINYA:
- SICA = Satu aplikasi katering
- SHGA = Satu aplikasi hamper  
- SCA  = Satu aplikasi analisis kontrak
- SMA  = App Store-nya semua Sovereign Agents

USER bisa:
1. Beli "SICA Agent" untuk kelola katering
2. Tambah "SHGA Agent" untuk hamper service
3. Tambah "SCA Agent" untuk review kontrak
4. Semuanya terhubung = FULL SOVEREIGN BUSINESS SUITE
```

## 🤖 TECHNICAL ARCHITECTURE SMA

### Agent Orchestration Layer
```typescript
// SMA Agent Orchestrator
interface AgentBlueprint {
  id: string
  name: string
  includes: AgentType[]
  tools: Tool[]
  connections: AgentConnection[]
}

interface AgentConnection {
  from_agent: AgentType
  to_agent: AgentType
  shared_data: string[] // ['customers', 'orders', 'analytics']
  trigger: 'manual' | 'automatic' | 'webhook'
}

// Example: SICA + SHGA connection
const sicaSHGABundle: AgentBlueprint = {
  id: 'sovereign-katering-pro',
  name: 'Sovereign Katering Pro',
  includes: ['SICA', 'SHGA'],
  tools: ['groq_ai', 'supabase', 'midtrans', 'whatsapp'],
  connections: [
    {
      from_agent: 'SICA',
      to_agent: 'SHGA',
      shared_data: ['customers', 'events'],
      trigger: 'automatic' // When SICA has event order → suggest SHGA hamper
    }
  ]
}
```

### Cross-Agent Data Sharing
```sql
-- When user has both SICA and SHGA:
-- Shared customer table (not duplicated)
CREATE TABLE sma_shared_customers (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  unified_business_id UUID,
  
  -- SICA link
  sica_business_id UUID REFERENCES sica_businesses(id),
  
  -- SHGA link
  shga_business_id UUID REFERENCES shga_businesses(id),
  
  -- Customer data (shared)
  name TEXT NOT NULL,
  phone TEXT,
  email TEXT,
  address TEXT,
  company TEXT,
  
  -- Aggregated data
  total_orders INTEGER DEFAULT 0,
  total_spent NUMERIC DEFAULT 0,
  last_order_date DATE,
  preferred_agent TEXT, -- 'sica' | 'shga' | 'both'
  
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```
