## 3. MULTI-AGENT WORKFLOW (THE PREDATOR LOOP)

1.  **Task Initiation:** Anda memberikan perintah di Master App (misal: "Target 10 Barber di Jakarta").
2.  **Scouting (The Scout):** Agen Scout melakukan scrape real-time dan memberikan daftar mentah.
3.  **Profiling (The Profiler):** Agen Profiler memfilter daftar tersebut dan memberikan skor 0-100. Hanya skor > 80 yang lanjut.
4.  **Content Creation (The Ghostwriter):** Agen Ghostwriter membuatkan pesan penawaran dan draft landing page.
5.  **Construction (The Architect):** Agen Architect membuat kode demo web dan melakukan **Auto-Deploy** ke Vercel/Cloudflare.
6.  **Closing (The Closer - Human-in-the-Loop):** SPBA menyiapkan link WA dan Demo. Anda tinggal klik "Send."
7.  **Harvesting (The Harvester):** Setelah deal, agen Harvester mencatat transaksi dan memindahkan profit ke Liquidity Fund.
