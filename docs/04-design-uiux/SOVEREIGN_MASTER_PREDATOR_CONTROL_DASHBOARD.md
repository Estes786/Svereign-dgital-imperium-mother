## 2. THE MASTER DASHBOARD (UI LAYOUT)

### 2.1 Tab 1: Predator (The Command Center)
- **Goal Setting:** Input "Target Profit" (misal: "$500") dan "Target Area" (misal: "Jakarta Selatan").
- **Crew Status:** Lihat status agen (Scout, Profiler, Ghostwriter, Architect, Harvester) yang sedang bekerja secara paralel.
- **Action:** Tombol "Launch Predator Loop" untuk memulai proses otonom.

### 2.2 Tab 2: Leads (The Hunter's Eye)
- **Hot Leads List:** Daftar bisnis dengan **AI Score (0-100)** dan **Digital Gap Analysis**.
- **Instant Demo:** Tombol "Generate & Deploy Demo" (otomatis via Vercel/Cloudflare API).
- **WA Outreach:** Tombol hijau "Send WA Hook" (WA Deeplink).

### 2.3 Tab 3: Treasury (The Harvester)
- **Revenue Tracker:** Grafik pendapatan real-time (Rupiah & USD).
- **Liquidity Progress:** Progress bar menuju target $500.
- **Auto-Staking:** Lihat status $HYPHA yang sedang di-stake untuk yield.

### 2.4 Tab 4: Logs (The Agent Brain)
- **Activity Log:** Lihat setiap keputusan yang diambil oleh CrewAI (misal: "Scout menemukan 50 leads," "Profiler memfilter 10 hot leads").
- **Error Handling:** Notifikasi jika ada API limit atau kegagalan deployment.
