# SOVEREIGN MASTER TOOL: INTERFACE & INTEGRATION FRAMEWORK

## 1. PENDAHULUAN: PUSAT KONTROL BERDAULAT
Sovereign Master Tool (SMT) adalah antarmuka tunggal (Single Interface) yang Anda gunakan untuk mengendalikan **Sovereign Personal Business Agent (SPBA)**. Desain ini berfokus pada **Mobile-First UX** agar Anda bisa menjalankan seluruh kerajaan bisnis (Eco-System) hanya dari genggaman tangan.

---

## 2. THE MASTER DASHBOARD (UI LAYOUT)

### 2.1 Tab 1: Hunter (The Lead Predator)
- **Search Bar:** Masukkan "Kategori Bisnis" + "Lokasi" (misal: "Barbershop Kemang").
- **Scraping Engine:** Tombol "Scrape Real Data" (via SerpAPI).
- **Lead Cards:** Daftar bisnis dengan Thumbnail, Rating, dan **AI Score (0-100)**.
- **Action:** Tombol "Save to Pipeline" atau "Quick Scan."

### 2.2 Tab 2: Pipeline (The Closer CRM)
- **Stages:** New → Contacted → Follow Up → Closed.
- **Lead Detail:** Klik untuk melihat "Digital Gap Analysis," "Suggested Service," dan "Price."
- **WA Action:** Tombol hijau "Send WA Script" yang otomatis membuka WhatsApp dengan pesan personal.

### 2.3 Tab 3: Builder (Instant Demo)
- **Demo Generator:** Tombol "Generate Ghost Website."
- **Preview Window:** Lihat tampilan website/PWA dalam 30 detik.
- **Share Link:** Link unik untuk dikirimkan ke klien sebagai bukti nyata (Proof of Concept).

### 2.4 Tab 4: Treasury (The Harvester)
- **Revenue Counter:** Total uang masuk hari ini/minggu ini.
- **Liquidity Progress:** Progress bar menuju target $500.
- **Profit Distribution:** Lihat alokasi untuk Operasional vs Likuiditas vs Staking.

## 4. TECH INTEGRATION (API & WEBHOOKS)
- **WhatsApp:** Deeplink (`wa.me`) + Template System.
- **Payments:** Webhook dari DOKU/Midtrans untuk auto-update status order.
- **Maps:** SerpAPI untuk real-time Google Maps scraping.
- **AI Brain:** Groq API (Llama-3.3-70b) untuk scoring dan script generation.
- **Storage:** Supabase untuk database dan aset visual.
