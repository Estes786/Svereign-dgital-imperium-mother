## 3. TEKNIK OPTIMASI RAM 2GB (GOD MODE)

Agar VPS kamu tidak *crash*, kita gunakan teknik **"Lean Execution"**:

*   **Runtime**: Gunakan **Bun** (bukan Node.js biasa) karena Bun 3x lebih hemat RAM.
*   **Process Manager**: Gunakan **PM2** dengan fitur `max-memory-restart` di 1.5GB. Jadi kalau ada kebocoran memori, sistem otomatis restart sendiri (Self-Healing).
*   **Swap Space**: Aktifkan 4GB Virtual RAM (Swap) di SSD VPS kamu sebagai cadangan darurat.
*   **Stateless Agents**: Pastikan Agent tidak menyimpan data di RAM, semua status harus ditulis ke Supabase.
